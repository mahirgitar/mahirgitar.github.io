14. harga gula 4/3 harga beras, harga beras 2/5 harga telur,selisih harga gula dn harga beras 2.400,berapa kh harga telur


Jawab:

Harga telu adalah Rp 7.200

Penjelasan dengan langkah-langkah:

Misa harga beras  =  b,

        harga telur  = t,

        harga gula  =  g,  maka:

g  =  4/3 b

b  =  2/5 t

   g - b   =  2.400

4/3b - b  =  2.400

     1/3b  =  2.400

         b  =  2.400 x 3

         b  =  7.200

b  =  2/5t

t  =  b x 5/2

t  =  7.200 x 5 / 2

t  =  18.000

Perbandingannya :
* gula : beras = 4 : 3 ( x2 )
* beras : telur = 2 : 5 ( x3 )

=> gula : beras : telur = 8 : 6 : 15

Harga telur = 15/2 × Rp 2.400 = Rp 18.000

pembuktian :
