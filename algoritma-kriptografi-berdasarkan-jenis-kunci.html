Algoritma Kriptografi
Kriptografi dapat diartikan sebagai ilmu atau seni untuk menjaga keamanan pesan. Pada prinsipnya kriptografi memiliki 4 komponen utama yaitu:

Plaintext, yaitu pesan yang dapat dibaca
Ciphertext, yaitu pesan acak yang tidka dapat dibaca
Key, yaitu kunci untuk melakukan teknik kriptografi
Algorithm, yaitu metode untuk melakukan enkrispi dan dekripsi

23. Algoritma yang mencari informasi berdasarkan kata kunci mulai dari awal array hingga akhir array adalah


Jawaban:

Salah satu algoritma yang dapat digunakan untuk mencari informasi berdasarkan kata kunci dari awal array hingga akhir array adalah algoritma Linear Search. Berikut adalah pseudocode untuk algoritma Linear Search:

```

1. Inisialisasi posisiAwal = -1 (berarti kata kunci tidak ditemukan)

2. Untuk setiap elemen arr[i] dalam array:

a. Jika arr[i] sama dengan kata kunci:

i. Set posisiAwal = i

ii. Hentikan pencarian

3. Kembalikan posisiAwal

```

Dalam algoritma ini, pencarian dimulai dari awal array dan berlanjut hingga akhir array. Jika kata kunci ditemukan, posisi awal diperbarui dengan indeks di mana kata kunci ditemukan. Jika kata kunci tidak ditemukan, posisiAwal tetap -1.

Berikut adalah contoh implementasi algoritma Linear Search dalam bahasa Python:

```python

def linear_search(arr, kata_kunci):

posisi_awal = -1

for i in range(len(arr)):

if arr[i] == kata_kunci:

posisi_awal = i

break

return posisi_awal

```

Anda dapat memanggil fungsi `linear_search` dengan memberikan array dan kata kunci yang ingin dicari sebagai argumen. Fungsi akan mengembalikan indeks pertama di mana kata kunci ditemukan, atau -1 jika kata kunci tidak ada dalam array.

Contoh penggunaan:

```python

array = [4, 2, 7, 1, 9, 5]

kata_kunci = 7

posisi = linear_search(array, kata_kunci)

print("Kata kunci ditemukan pada indeks:", posisi)

```

Output:

```

Kata kunci ditemukan pada indeks: 2

```

Dalam contoh di atas, kata kunci "7" ditemukan pada indeks ke-2 dalam array.

32. Berdasarkan model kunci, kriptografi dapat dibedakan menjadi 3 yaitu


Jawaban:

Model Kritografi :

1.Kriptografi Simetris => algoritma yang menggunakan kunci yang sama untuk proses enkripsi dan proses deskripsi. Algoritma kriptografi simetris dibagi menjadi dua kategori yaitu algoritma aliran (Stream Ciphers) dan algoritma blok (Block Ciphers).

2. Kriptografi Asimetris => algoritma yang menggunakan kunci yang berbeda untuk proses enkripsi dan deskripsi. Dimana kunci enkripsi dapat disebarkan kepada umum dan dinamakan sebagai kunci publik (public key), sedangkan kunci deskripsi disimpan untuk digunakan sendiri dan dinamakan sebagai kunci pribadi (private key).

40. kunci dari sebuah algoritma serta bernilai benar atau salah adalah​


Jawaban:

Kunci dari sebuah algoritma serta apakah algoritma tersebut bernilai benar atau salah tergantung pada beberapa faktor, termasuk:

1. **Ketepatan Logika Algoritma:** Algoritma harus dirancang dengan benar dan logis sehingga dapat melakukan tugas yang dimaksud dengan benar. Jika logika algoritma itu sendiri salah, maka hasilnya juga akan salah.

2. **Implementasi yang Tepat:** Algoritma harus diimplementasikan dengan benar dalam bahasa pemrograman tertentu. Kesalahan dalam implementasi dapat mengakibatkan hasil yang tidak sesuai dengan yang diharapkan.

3. **Data Input yang Valid:** Algoritma mungkin memerlukan data input. Data yang salah atau tidak valid dapat menghasilkan keluaran yang salah.

4. **Kondisi Awal yang Benar:** Beberapa algoritma mungkin memerlukan kondisi awal tertentu. Jika kondisi awal tidak benar, hasilnya bisa salah.

5. **Tes dan Verifikasi:** Untuk menentukan apakah sebuah algoritma berfungsi dengan benar, Anda perlu melakukan pengujian dan verifikasi. Ini melibatkan memberikan masukan yang berbeda dan memastikan hasilnya sesuai dengan yang diharapkan.
