Di dalam java, percabangan yang dituliskan ada tiga yaitu:

- if

- if...else

- switch...case

Perintah if dapat dituliskan jika kondisi memenuhi syarat maka pernyataan akan dikerjakan dan jika tidak memenuhi syarat maka program akan di batalkan.

if (kondisi);

Perintah if...else dapat dituliskan jika kondisi memenuhi salah satu dari banyak syarat maka, salah satu pernyataan akan dikerjakan dan jika tidak memenuhi syarat sama sekali maka, program akan dibatalkan.

Perintah switch..case dapat dituliskan jika kondisi memenuhi salah satu dari banyak syarat maka, salah satu pernyataan akan dikerjakan dan jika tidak memenuhi syarat sama sekali maka, program akan dibatalkan. switch (ekspresi) {

- while

- do...while

- for

while (kondisi) {

Perintah do...while dapat dituliskan sepeti while, akan tetapi jika do...while pengecekan kondisi (syarat) perulangan dilakukan setelah eksekusi statement yang diulang. Sehingga statement dalam blok do-while paling sedikit dieksekusi satu kali.

do {

3. Apa kegunaan struktur kontrol percabangan?


Kegunaan struktur kontrol percabangan dalam untuk menambah variasi dalam program seperti menyediakan pilihan-pilihan didalamnya yang nantinya akan digunakan oleh pengguna. Struktur kontrol percabangan memang memerlukan logika lebih terutama untuk program kompleks, oleh karena itu hal ini dapat membuat suatu program memiliki nilai jual tinggi.

Pembahasan

Struktur kontrol percabangan dalam bahasa pemrograman dikenal dengan dua perintah yaitu IF-ELSE dan SWITCH.  Keduanya memiliki kelebihan dan kekurangan dalam penggunaanya walau sama-sama lebih kompleks. Dalam hal efisiensi switch memang lebih efisien namun, penggunaanya jarang dalam  program yang ada  sekarang karena memiliki beberapa keterbatasan. Penggunaan IF-ELSE masih lebih banyak dan lebih fleksibel sehingga lebih dipilih oleh banyak developer dan programmer.

7. Uraikan yang kita ketahui tentang struktur kontrol percabangan.


Jawaban:

Struktur kontrol percabangan adalah suatu konsep dalam pemrograman yang memungkinkan programmer untuk mengendalikan alur eksekusi program berdasarkan suatu kondisi tertentu. Ada beberapa jenis struktur kontrol percabangan yang dikenal, di antaranya adalah if-then-else, switch-case, dan conditional operator.

If-then-else adalah struktur percabangan yang paling dasar. Struktur ini mengevaluasi sebuah kondisi, dan jika kondisi tersebut bernilai benar, maka bagian kode yang ada di dalam blok "then" akan dijalankan. Namun, jika kondisi tersebut bernilai salah, maka bagian kode yang ada di dalam blok "else" akan dijalankan.

Switch-case adalah struktur percabangan yang sering digunakan untuk mengevaluasi banyak kondisi yang berbeda. Struktur ini mengevaluasi sebuah variabel, dan mengeksekusi bagian kode yang sesuai dengan nilai dari variabel tersebut.

Conditional operator adalah struktur percabangan yang lebih singkat dan lebih fleksibel dibandingkan dengan if-then-else. Struktur ini mengevaluasi sebuah kondisi, dan mengembalikan salah satu dari dua nilai yang telah ditentukan, tergantung pada hasil evaluasi kondisi tersebut.

Secara umum, struktur kontrol percabangan digunakan untuk membuat alur program yang lebih dinamis dan fleksibel, sehingga dalam mengeksekusi suatu program, sebuah alur eksekusi dapat memiliki lebih dari satu jalur yang berbeda sesuai dengan kondisi yang terjadi. Struktur kontrol percabangan memungkinkan program untuk memilih jalur eksekusi yang sesuai dengan kondisi yang ditentukan, sehingga program dapat beradaptasi dengan situasi yang berbeda. Struktur kontrol percabangan juga dapat membantu program untuk meminimalkan kode yang berulang-ulang, sehingga program menjadi lebih terorganisir dan mudah dibaca.

Blok

Blok adalah pernyataan sederhana yang dimulai dengan { dan diakhiri dengan }. Tujuannya untuk mengelompokkan beberapa perintah sehingga lebih dimengerti. Misalnya

{

perintah

Pernyataan blok biasanya terdapat dalam struktur kontrol, tetapi sebetulnya tanpa struktur kontrol pun, blok tetap berlaku. Contoh sederhana penggunaan blok ini adalah dalam subrutin main() yang kita lakukan pada contoh-contoh sebelumnya.

Berikut ini adalah 2 contoh penggunaan blok.

{

System.out.println("Hallo");

System.out.println("Dunia");

}

{ // blok ini digunakan untuk menukar isi variable x dan y

x = y;

Perulangan

Pengulangan adalah suatu proses didalam program yang dapat menegeksekusi satu ( beberapa ) statemen yang sama secara berulang sampai ditemukan kondisi untuk berhenti. Dalam java terdapat tiga buah jenis struktur pengulangan, yaitu for, while, dan do while.

v Strukur for

Bentuk umum penulisa sintaks untuk membangun struktur pengulangan for

for(insialisasi;kondisi;iterasi); {

statement 1 ;

statement 2;

}

Proses pengulangan akan terus dilakukan selama kondisi menghasilkan true. Dengan kata lain, proses pengulangan hanya akan dihentikan apabila kondisi telah bernilai false atau sudah tidak terpenuhi lagi.

Contoh program :

class for1 {

public static void main (String[] args) {

for ( int i = 0 ; i < 10 ; i++ ) {

System.out.println(“java”); }

}

}

Maka outputnya akan tercetak teks java sebanyak 10 kali.

insialisasi

Program akan menarik jika dibutuhkan persyaratan tertentu untuk menjalankan suatu pernyataan, dan melewati pernyataan lainnya. Java menyediakan sejumlah mekanisme untuk menghasilkan pengaturan bagian program mana yang dijalankan, berupa pernyataan pernyataan percabangan . terdapat beberapa jenis percabangan di java yaitu if, if-else, dan switch.

v Percabangan if

Percabangan if di java menyatakan pernyataan akan diekseskusi jika memnuhi syarat atau kondisi tertentu. Sintaks percabangan ini adalah sebagai berikut :

If(kondisi){

Statemen1;

Statemen2;

...

}

}

}

Output nya : nilai a lebih kecil dari 5

v Percabangan if else

Percabangan ini untuk memilih salah 1 dari 2 kemungkinan kemunculan. Dengan kata lain, bentuk if else menyebabkan eksekusi dijalankan melalui sekumpulan boolean, sehingga hanya bagian tertentu program yang dijalankan. Berikut bentuk umum pernyataan if else :

If (kondisi)

Statement jika kondisi benar

Else

Statement jka kondisi salah

Contoh program :

class duakondisi {

public static void main (String[] args) {

System.out.println(a + “ lebih kecil dari 5”);

}

else {

System.out.println(a + “ lebih besar dari 5 “ );

}

if ( b < 5 ) {

System.out.println(b + “ lebih kecil dari 5”);

}

else {

System.out.println(b + “ lebih besar dari 5 “ );

}

}

}

1 lebih kecil dari 5

10 lebih besar dari 5

switch (expression) {

case value1;

break;

case value2;

break;

case valueN;

break;

default;

}

C

for (int i = 1; i <= 10; i++) {

   cout << i << " ";

C

}

// Output: Lulus
